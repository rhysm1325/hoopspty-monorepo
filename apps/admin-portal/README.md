# HoopsPty Admin Portal

This will be the administrative portal for managing all business operations.

## Features (Planned)
- User management across all apps
- System configuration
- Cross-app reporting
- Business analytics dashboard

## Tech Stack (Planned)
- Next.js 14
- Shared auth from @hoops-pty/shared-auth
- Shared UI components from @hoops-pty/shared-ui
